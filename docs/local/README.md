# 本地过程文档入口

`docs/local/` 用于保存本地过程文档与临时记录。其余内容默认不入库；如确有协作需要，可阶段性入库，并在生命周期结束后删除或迁出。

## 建议子目录

- `decisions/`
  - 尚未固化为正式文档的决策草稿
- `notes/`
  - 调研记录、命令摘录、零散笔记
- `sessions/`
  - 按日期归档的会话过程记录
- `tracking/`
  - 阶段计划、当前进度、下一步动作与阻塞项

## 使用规则

1. 本目录用于本地记忆和排错，不作为正式结论来源。
2. 每次记录结束补充“当前停点 / 下一步”。
3. 如内容已稳定，应迁移到 `docs/public/`。
4. 如内容为阶段性入库文档，应在生命周期结束后删除或迁出。
